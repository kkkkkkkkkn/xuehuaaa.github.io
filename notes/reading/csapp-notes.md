## 第一章：计算机系统漫游

### 关键概念

1. **信息 = 位 + 上下文**
   - 计算机中的所有信息都是位（0/1）的序列
   - 同样的字节序列，在不同的上下文中表示不同的含义

2. **程序翻译过程**
   ```
   源代码 (hello.c)
       ↓ 预处理 (cpp)
    hello.i (修改后的源代码)
       ↓ 编译 (ccl)
    hello.s (汇编程序)
       ↓ 汇编 (as)
    hello.o (可重定位目标程序)
       ↓ 链接 (ld)
    hello (可执行目标程序)
   ```

3. **系统的硬件组成**
   - CPU（中央处理单元）
   - 主存（内存）
   - I/O 设备
   - 系统总线

### 重要启示

> "程序员需要理解计算机系统的底层工作原理，才能写出高效、可靠的程序。"

## 第二章：信息的表示和处理

### 2.1 信息存储

#### 十六进制表示

- 二进制 ↔ 十六进制：每 4 位二进制对应 1 位十六进制
- 示例：`0b1010` = `0xA`

#### 字长

- 字长决定了系统虚拟地址的范围
- 32 位系统：最大内存 4GB（2^32 字节）
- 64 位系统：理论最大内存 16EB（2^64 字节）

### 2.2 整数表示

#### 无符号整数

- 范围：`[0, 2^w - 1]`，其中 w 是位数

#### 有符号整数（补码表示）

- 范围：`[-2^(w-1), 2^(w-1) - 1]`
- 补码的最高位表示符号（0 正 1 负）
- 负数的补码 = 正数的补码取反 + 1

```c
// 示例：-1 的 32 位补码表示
// 1 的补码:   00000000 00000000 00000000 00000001
// 取反:       11111111 11111111 11111111 11111110
// 加 1:       11111111 11111111 11111111 11111111
// 即 0xFFFFFFFF
```

### 2.3 浮点数表示

#### IEEE 754 标准

```
单精度 (32位): [符号位 1] [指数 8] [尾数 23]
双精度 (64位): [符号位 1] [指数 11] [尾数 52]
```

#### 重要特性

1. 浮点数不是连续的，存在精度限制
2. `(x + y) + z` 不一定等于 `x + (y + z)`（不满足结合律）
3. 避免对浮点数使用 `==` 比较，应使用误差范围

```c
// 错误的比较方式
if (a == b) { ... }  // 可能失败！

// 正确的比较方式
if (fabs(a - b) < 1e-9) { ... }
```

## 第三章：程序的机器级表示

### 3.1 汇编基础

#### x86-64 寄存器

| 寄存器 | 用途 |
|--------|------|
| %rax   | 返回值 |
| %rbx   | 被调用者保存 |
| %rcx   | 第 4 个参数 |
| %rdx   | 第 3 个参数 |
| %rsi   | 第 2 个参数 |
| %rdi   | 第 1 个参数 |
| %rbp   | 帧指针 |
| %rsp   | 栈指针 |

### 3.2 控制结构

#### 条件跳转

```assembly
# if (a > b) { ... }
cmpq %rbx, %rax    # 比较 a 和 b
jle  .L1            # 如果 a <= b，跳过
...                 # a > b 时的代码
.L1:
```

#### 循环

```c
// for 循环
for (int i = 0; i < n; i++) {
    sum += a[i];
}
```

```assembly
# 对应的汇编
movl $0, %ecx       # i = 0
jmp  .L2
.L3:
    # 循环体
    ...
    addl $1, %ecx   # i++
.L2:
    cmpl %rdx, %ecx # i < n ?
    jl   .L3        # 如果 i < n，继续循环
```

## 第九章：虚拟内存

### 关键概念

1. **虚拟内存的作用**
   - 为每个进程提供独立的地址空间
   - 简化内存管理
   - 提供内存保护

2. **地址翻译**
   ```
   虚拟地址 (VA) → [MMU] → 物理地址 (PA)
   ```

3. **页表**
   - 页表存储在主存中
   - 每个进程有自己的页表
   - TLB（快表）缓存常用的页表项

### 重要公式

```
VP: 虚拟页
PP: 物理页（页帧）

虚拟地址空间大小 = 2^v
物理地址空间大小 = 2^p
```
